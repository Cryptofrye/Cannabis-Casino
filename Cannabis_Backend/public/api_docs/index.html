    Introduction

    새로운 온라인 강의 서비스 누림(nurim) API 문서 입니다.

    Authentication

    signIn

    HTTP Request

    POST http://example.com/auth/signIn

    Request Type
    id String(20)
    password String(30)

    signUp

    HTTP Request

    POST http://example.com/auth/signUp

    Request Type
    id String(20)
    password String(30)
    email String(30)
    success : 200 failure : 404, 500

    Courses

    Get All Courses

    [
  {
    "title": "python 강의",
    "intro": "파이썬을 갈켜드립니다",
    "people": "15", 
    "img": "example.com/python.png",
    "category": "programming"
  },
  {
      "title": "java 강의",
      "intro": "자바를 갈켜드립니다",
      "people": "10",

      "category": "programming"
    }
]

    HTTP Request

    GET http://example.com/courses/

    강의를 수강자가 많은 순으로 가져온다.
    Remember — a happy kitten is an authenticated kitten!

    Get a Specific Course

    특정 강의 하나의 간략한 정보를 가져온다.

    {
      "title": "python 강의",
      "intro": "파이썬을 갈켜드립니다",
      "people": "15", 
      "img": "example.com/python.png",
      "category": "programming"
}

    HTTP Request

    GET http://example.com/courses/<title>

    Request Type
    title String(20)

    Get Course Comment

    특정 강의 하나의 모든 댓글 정보를 가져온다.

    [
      {
          "content": "so easy!",
          "user": "hue williams"
      },
      {
          "content": "야발",
          "user": "junu"
      }
]

    HTTP Request

    GET http://example.com/courses/comment/<title>

    Request Type
    title String(20)

    Post Course Comment

    특정 강의에 댓글을 추가한다.

    HTTP Request

    POST http://example.com/courses/comment/<title>

    Request Type
    content string
    user string

    Errors

    This error section is stored in a separate file in includes/_errors.md. Slate allows you to optionally separate out your docs into many files...just save them to the includes folder and add them to the top of your index.md's frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks
    401 Unauthorized -- Your API key is wrong
    403 Forbidden -- The kitten requested is hidden for administrators only
    404 Not Found -- The specified kitten could not be found
    405 Method Not Allowed -- You tried to access a kitten with an invalid method
    406 Not Acceptable -- You requested a format that isn't json
    410 Gone -- The kitten requested has been removed from our servers
    418 I'm a teapot
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarially offline for maintanance. Please try again later.
